# UI & Image Generation Guidelines
When generating images for articles, case studies, or the site in general, STRICTLY adhere to the following prompt style:
"Minimalist monochrome infographic illustration, [TOPIC], clean light grey and white background, simple elegant lines, wireframe style, subtle beige or pale orange accent color, a lot of white space, clean corporate design, 1:1 ratio, no text."
Never generate colorful, abstract, or highly detailed images unless explicitly asked. Stick to this clean, monochrome, infographic wireframe style.
